welcome to the git area :3

-To commit to your local branche enter: git commit -am "commit message"

-To push your commit to the world wide web guy just like spider man is a guy. enter: git push origin main

-If working with others you may have to pull in order to push as someone may have made changes: git pull origin main

-Origin is the shorthand for the remote repo who is located far away in the world wide web guy. Like spiderman.

-Main is an example of your current branche. The command is as follows: $ git push "remote" "branch"
